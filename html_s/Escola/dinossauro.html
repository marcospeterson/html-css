    Oque é HTML?

    O HTML, conhecido como Linguagem de Marcação de Hipertexto,(HyperText Markup Language) é a base fundamental para construir páginas da web e aplicativos online. Ele usa uma série de marcações para definir a estrutura e o estilo de um documento. Quando falamos de hipertexto, estamos nos referindo a textos que têm links para outros textos, criando uma experiência de navegação conectada. Ao contrário das linguagens de programação, o HTML não tem o poder de criar funcionalidades dinâmicas. Sua função principal é organizar o conteúdo da página, permitindo que os usuários criem seções, parágrafos e links por meio de elementos, tags e atributos. Embora seja essencial para apresentar e organizar informações na web, para funcionalidades mais avançadas e interações dinâmicas, geralmente é combinado com outras tecnologias, como CSS (para estilos) e JavaScript (para interatividade).

    A estruta básica do html segue o seguinte padrão:

    <!DOCTYPE html>
    <html lang="pt-br">
    <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Tags HTML</title>
    </head>
    <body>
    </body>
    </html>

    <!DOCTYPE html>: Esta é uma declaração do tipo de documento (Document Type Declaration - DTD) para indicar ao navegador que o documento HTML está seguindo a especificação HTML5.

    X<html lang="pt-br">: Abre o elemento raiz do documento HTML. O atributo lang é usado para indicar o idioma do documento. Neste caso, "pt-br" representa o português brasileiro.

    <head>: Contém metadados sobre o documento, como o conjunto de caracteres utilizado, configurações de viewport (para tornar o site responsivo) e o título da página.

    <meta charset="UTF-8">: Especifica que o conjunto de caracteres UTF-8 deve ser usado para interpretar o conteúdo da página. Isso é importante para garantir que caracteres especiais e acentos sejam exibidos corretamente.

    <meta name="viewport" content="width=device-width, initial-scale=1.0">: Configurações relacionadas à visualização em dispositivos móveis. width=device-width significa que a largura da tela deve ser igual à largura do dispositivo, e initial-scale=1.0 define o nível de zoom inicial.

    <title>Tags HTML: Define o título da página que será exibido na aba do navegador ou na barra de título.

    <body>: Contém o conteúdo real da página, como texto, imagens, links, etc. <body> e </body>.

    Você Sabia?

    O HTML foi inventado por Tim Berners-Lee, um físico do centro de pesquisas CERN, na Suíça. Ele surgiu com a ideia de um sistema de hipertexto na internet. Hipertexto significa um texto que possui referências (links) para outros textos que podem ser acessados imediatamente.
